## Git 使用方法

```bash
# HTTPS 方式
git clone https://github.com/用户名/仓库名.git

# SSH 方式
git clone git@github.com:用户名/仓库名.git

# 克隆到指定目录
git clone https://github.com/用户名/仓库名.git 目标文件夹名
```

## 如果你没有 Git
### Windows


```bash
# winget 或者在 Microsoft Store 里搜索git 
winget install git

# Scoop
scoop install git

# Chocolatey
choco install git
```

> 没有包管理器？[git-scm.com](https://git-scm.com/download/win) 下载安装包

### Linux

```bash
# Debian / Ubuntu
sudo apt install git

# CentOS / RHEL 7
sudo yum install git

# CentOS / RHEL 8+ / Fedora
sudo dnf install git

# Arch Linux
sudo pacman -S git

# openSUSE
sudo zypper install git

# Alpine
sudo apk add git

# Gentoo
sudo emerge --ask dev-vcs/git

# Solus
sudo eopkg install git

# Void Linux
sudo xbps-install -S git
```

## 基础要求
1. Kotlin 的基础格式（不会可以问 AI）
2. Minecraft 里的类型名（不会也可以问 AI）
3. 会添加 Gradle 依赖（Gradle 脚本均已拆分在 `scripts` 文件夹下）
4. 会组织语言，写代码
   - 语言：Kotlin / Java
   - Java 用于写 Mixin（不会直接叫 AI 写）([e.g.](../../src/main/java/rain/gtetcore/GTET/mixin/MixinFixGTBlocksNpe.java))
5. 主要部分请写注释，若过多请叫 AI 在类上面添加对照表
6. <s>用AI开发时请注意自己的token额度</s>
