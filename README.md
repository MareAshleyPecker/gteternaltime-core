# GregTech Eternal Time

一个还在做的整合包 / A modpack still in development.
___

# 主要 / Overview

1. 以格雷科技为主的一个整合包，当前目标是做出UV电压之前的内容，本包不求太难
   - A GregTech-focused modpack, currently aiming to cover content up to UV voltage; not intended to be overly difficult
2. 主线以适中难度(后续可能会添加专家或者坐牢难度)
   - Moderate difficulty for the main progression (expert/painful mode may be added later)
3. 每个电压添加优化游戏体验的机器
   - Quality-of-life machines added at each voltage tier
4. 计划完成到MAX电压等级
   - Planned to reach MAX voltage tier
5. [相关计划](README/zh_CN/计划.md) / [Plans](README/en_US/plan.md)
   - 还没想好，目前还在打地基 / Still brainstorming, currently laying the groundwork
